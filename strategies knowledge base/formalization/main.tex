\textbf{Document intent.}
This report translates the repository's discretionary trading materials into explicit research language \emph{and} into an operational trading playbook.
It does not claim that every source idea is correct, current, or robust.
Its purpose is to preserve the human logic, expose ambiguity, show how a trader would actually search for and qualify each setup in real time, and map each idea into testable components for logging, replay, feature engineering, rule-based detection, and later ML overlays.
